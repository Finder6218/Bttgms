# 公交汽车票款综合管理系统

《公交汽车票款综合管理系统》是为了方便公交工作人员能够有效的管理公交票款，对每辆车的维修，投保，年审，违章，加油，投诉等进行全面管理的一套管理软件。  
软件对无人售票公交和有人售票公交进行票款分别管理，还可以对每个月上班拿票、下班交款及无人票款登记的业务记录进行统一月结。  
软件功能包括：  
1、基础数据（公交线路设置，票据类型，车辆信息，驾驶员信息，售票员信息及其他员工信息）。  
2、有人售票管理（上班拿票，下班交款，票款月结，上班拿票查询统计，下班交款统计查询，月结拿票查询统计，月结交款查询统计）。  
3、无人售票管理（票款登记，票款查询，票款月结）。  
4、综合管理（维修记录，投保记录，年审记录，违章记录，投诉记录，加油记录，费用支出，费用收入）。  
5、查询报表（车辆信息查询，维修记录查询，投保记录查询，年审记录查询，事故记录查询，违章记录查询，投诉记录查询，出车记录查询，费用支出查询，费用收入查询，加油记录查询，员工记录查询）。  
本系统界面更加友好，操作更加人性化，使用更加方便，让您的工作更加得心应手；方便快捷的查询功能。
